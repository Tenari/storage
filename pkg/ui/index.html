Telegram Bot Configuration

Open Telegram and search for "@BotFather".

Start a conversation and type `/newbot`.

Follow prompts to create a new bot.

Securely copy the API key displayed.

Paste the bot API key into the input below.

For OpenAI Whisper and LLM functionality, an OpenAI API key is optional:

  1. Visit OpenAI's signup page.
  2. Create an account and navigate to the API section.
  3. Click "New API Key" to generate a key.
  4. Securely copy the API key displayed.
  5. Type 'y' to proceed with entering the API key.
  6. Paste the API key into the input below.
